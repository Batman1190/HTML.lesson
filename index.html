🎨 Lesson 1: What is HTML?

HTML is like building blocks for websites! Just like LEGO bricks, we use HTML to build different parts of a website.

HTML stands for HyperText Markup Language. But don't worry about the big name - just remember it helps us build websites! 🏗️

🎈 Fun Fact: Every website you see on the internet uses HTML!

🎯 Exercise 1: Your First HTML!

Type this code exactly as you see it:

<h1>Hello World!</h1>
Hint: Make sure to type the < and > symbols. The words go between them!

🏷️ Lesson 2: HTML Tags

HTML tags are like labels that tell the computer what each part is!

Most tags come in pairs:

  • An opening tag: <p>
  • Your content in the middle
  • A closing tag: </p> (notice the /)
🎈 Fun Fact: Tags are like sandwiches - they have a top and bottom with yummy content in between!

🎯 Exercise 2: Make a Paragraph!

Create a paragraph that says: "I am learning HTML!"

<p>Your text here</p>
Hint: Use <p> to start and </p> to end. Put your sentence in between!

📏 Lesson 3: Headings - Big and Small!

Headings are like titles in a book. They help organize your webpage!

HTML has 6 sizes of headings:

  • <h1> - The BIGGEST! 🐘
  • <h2> - A bit smaller 🐕
  • <h3> - Medium size 🐈
  • <h4> - Getting smaller 🐁
  • <h5> - Very small 🐜
  • <h6> - The smallest 🦗
🎈 Fun Fact: h1 is like the title of a story, and h2 is like chapter names!

🎯 Exercise 3: Create a Big Heading!

Create an h2 heading that says: "My Awesome Website"

Hint: Use <h2> to start and </h2> to end. Don't forget the closing tag!

📝 Lesson 4: Making Lists!

Lists help you organize things, like your favorite toys or foods! 🍕🍦🍪

There are two types of lists:

  • Unordered Lists (with bullet points): <ul>
  • Ordered Lists (with numbers): <ol>

Each item in a list uses: <li> (li means "list item")

Example: 
<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

🎯 Exercise 4: Make a List!

Create an unordered list with two items: "Dogs" and "Cats"

Hint: Start with <ul>, then add two <li> items, and end with </ul>!

🖼️ Lesson 5: Adding Pictures!

Images make websites fun and colorful! 🌈

The image tag is special - it doesn't need a closing tag!

We use: <img src="picture.jpg" alt="description">

  • src tells where the picture is
  • alt is a description (helpful if the picture doesn't load!)
🎈 Fun Fact: The img tag is self-closing - it doesn't need a partner tag!

🎯 Exercise 5: Add an Image!

Create an image tag with src="https://via.placeholder.com/150" and alt="A placeholder image"

Hint: Remember: <img src="URL" alt="description"> - no closing tag needed!
